# AI Development Team Template

Copy this folder into a client/project repo, fill `docs/PROJECT_BRIEF.md`, then run the team loop.

## Recommended First Sequence

1. Project Manager: create PRD.
2. Solution Architect: create architecture and validation contract.
3. Lead Integrator: expand implementation queue.
4. Frontend/Backend/UX workers: implement small tasks.
5. QA + Security: validate.
6. DevOps: prepare deployment.
7. Lead Integrator: final release handoff.

## What This Uses From the Main Brain

- Orchestrator/worker/validator roles from [[Multi-Agent Systems That Ship]].
- Validation contracts and structured handoffs.
- Bounded loops with triggers, state, verification, stop conditions, and hard stops from [[Agent Loops and Durable Cron Jobs]].
- AX/harness ideas from [[Agentic Engineering]].
- Practical rules/skills/automation/cloud/worktree patterns from recent YouTube source ingests.
