# Skill: Backend Implementation

Use for one scoped backend/API/integration task.

## Procedure

1. Read assigned task, architecture, and env var docs.
2. Implement validation, error handling, auth/authorization, persistence, or integration logic as scoped.
3. Never commit secrets.
4. Add/update tests where appropriate.
5. Document new env vars or setup steps.

## Verification

Relevant tests/build pass; inputs are validated; errors are handled; no secret leakage.
