# Skill: Frontend Implementation

Use for one scoped UI task.

## Procedure

1. Read assigned task, PRD, architecture, and design/content rules.
2. Implement only the assigned UI slice.
3. Include responsive behavior, loading/empty/error states, labels, focus states, and accessible semantics.
4. Run assigned checks.
5. Record screenshots/browser evidence when possible.

## Verification

Task acceptance criteria met; relevant lint/typecheck/build/browser smoke checks pass or failures are documented.
