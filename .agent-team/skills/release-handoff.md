# Skill: Release Handoff

Use at the end of a loop batch or before human approval.

## Procedure

1. Summarize shipped scope against PRD.
2. List verification evidence and command results.
3. List unresolved limitations and follow-up tasks.
4. Confirm production readiness checklist status.
5. Request explicit human approval for deployment/client-visible release if needed.

## Verification

Handoff is clear enough for Luis/client to decide approve, reject, or request changes.
